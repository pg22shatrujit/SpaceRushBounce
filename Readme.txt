###**PG22 T2 Unity 1 - A3: Simple Game**
Submitted by: Shatrujit Aditya Kumar
Date: 14 March, 2022
Version: 1.0
----------
Flappy birds-style game with space ships

####**Download/Install**
---------
 - Extract and run the Unity project

####**Controls**
--------
 - Jump input (Space Bar) to make the player's ship  bounce up
 - Avoid other ships to increase score
 - Hitting a ship causes health to drop
